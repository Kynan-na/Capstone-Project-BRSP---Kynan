# Capstone-Project-BRSP---KynanA. Deskripsi Project
Capstone Project merupakan proyek akhir yang dirancang untuk mengintegrasikan seluruh pembelajaran selama program berlangsung. Peserta diminta untuk menyusun project analisis transcriptomics secara mandiri, mulai dari perumusan masalah hingga interpretasi hasil dan dokumentasi akhir.

B. Ketentuan Umum
Peserta diberikan kebebasan dalam menentukan komponen project, dengan ketentuan:
1. Peserta bebas memilih topik.
2. Peserta bebas memilih dataset transcriptomics.
3. Peserta bebas memilih paper pendukung sebagai dasar literatur atau pembanding.
4. Peserta mengerjakan project menggunakan komponen berikut:
    - Problem Statement (dirancang sejak Week 1)
    - Metode Analisis (dikembangkan dan diperdalam pada Week 2–3)

C. Output Akhir
Peserta wajib mengumpulkan output berikut:
1) Repository GitHub
Repository GitHub peserta harus memuat:
    - Script analisis
    - Dataset 
    - Visualisasi hasil
    - Dokumentasi dalam bentuk:
       a. README, dan/atau
       b. laporan singkat
2) Laporan Singkat
Laporan singkat dibuat menggunakan Bahasa Indonesia atau Bahasa Inggris, dengan struktur minimal:
   - Pendahuluan
   - Metode
   - Hasil dan Interpretasi
   - Kesimpulan
3) Mini Presentasi (Video)
Peserta membuat mini presentasi hasil dalam bentuk video dengan ketentuan:
   - Durasi maksimal 7 menit
   - Berisi ringkasan:
      a. topik & problem statement
      b. dataset dan metode yang digunakan
      c. hasil utama & interpretasi
      d. kesimpulan singkat
D. Ketentuan Pengumpulan
Tenggat waktu pengumpulan yaitu pada Minggu, 8 Maret 2026.
